# Pillventory

Pillventory is an Android mobile application aimed at that pharmaceutical industry with the goal of speeding up the counting and verification process of filling prescriptions or taking inventory. The application will allow the user to take a picture with the device’s camera or upload an image from the device’s gallery to process for a reliable pill count.
